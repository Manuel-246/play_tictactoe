# 🎮 Tic Tac Toe Infinity

A modern and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript** with a futuristic cyber-themed design.

![Tic Tac Toe Banner](tictactoe.jpeg)

## ✨ Features

### 🎯 Game Modes
- 👥 Player vs Player
- 🤖 Player vs AI

### 🎲 Coin Toss System
- Random coin toss decides who starts first.
- Fair gameplay every match.

### ⚙️ AI Difficulty Selection
- Easy
- Medium
- Impossible

### 🎨 Modern UI
- Futuristic cyber-themed background
- Animated pop-up notifications
- Interactive buttons with hover effects
- Responsive game board

### 🏆 Gameplay Features
- Win detection
- Draw detection
- Restart game option
- Home screen navigation
- Victory animations

## 🎮 How to Play

### Player vs AI
1. Click **Play VS AI**
2. Choose your symbol:
   - ❌ X
   - ⭕ O
3. Select difficulty level
4. Click **Start Match**
5. Coin toss decides who plays first

### Player vs Player
1. Click **Play VS Player**
2. Coin toss decides starting player
3. Alternate turns until someone wins
